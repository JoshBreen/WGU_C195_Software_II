BreenScheduling Program

Used to view upcoming schedule of appointments. Add appointments, and customers. Modify appointments and customers.

Josh Breen
Version 1.019
07/12/2021

IntelliJ IDEA 2021.1.3 (Community Edition)
JavaFX-SDK-11.0.2
Java SE 11.0.10

Directions:
User logs into the program. On the main window the top table shows the schedule and customers. Those are modified through
the buttons at the top. The bottom table shows the 3 reports. Those are controlled via the buttons to the side of the table.

The additional report shows all appointments that have past so the user could review what appointments had already
happened and with who.

MySQL Connector driver version number: mysql-connector-java:8.0.22
